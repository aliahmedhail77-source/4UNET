# تطبيق Android: إرسال الكروت تلقائيًا - V3 UI

هذه نسخة واجهات محسنة وليست نفس الواجهة القديمة.

## ما الجديد في الشكل؟

- شاشة رئيسية داخل مربعات Cards واضحة.
- إحصائيات في مربعات: الكروت المتاحة، المباعة، العمليات، المعلقة.
- فئات افتراضية مرتبة داخل بطاقات:
  50، 100، 150، 200، 250، 300، 500.
- صفحة استيراد منظمة داخل مربعات.
- صفحة سجلات منظمة داخل مربعات حالة.
- صفحة إعدادات منظمة.
- صفحة الأسماء الموثوقة لوَن كاش منظمة داخل بطاقات.
- دعم ONE Cash بمطابقة الاسم الثلاثي.
- إمكانية إضافة فئات أخرى.

## مهم

إذا ظهر التطبيق بنفس الواجهة القديمة بعد تثبيت APK:
1. تأكد أنك رفعت ملف MainActivity.java الجديد الموجود في هذا المشروع.
2. انتظر GitHub Actions حتى يبني APK جديد.
3. احذف التطبيق القديم من الهاتف.
4. ثبّت APK الجديد.
5. افتح التطبيق وستظهر نسخة الواجهة V3.
